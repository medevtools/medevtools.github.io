Politique de Confidentialité - TOEIC Prep Assistant

Dernière mise à jour : 3 décembre 2025

Introduction

TOEIC Prep Assistant ("l'Extension") est une extension Chrome conçue pour aider les utilisateurs à améliorer leur anglais et préparer le test TOEIC. L'extension propose un assistant IA conversationnel pour répondre aux questions de grammaire, vocabulaire et préparation TOEIC. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données.

Fonctionnalités principales

  • Chat IA : Posez des questions sur la grammaire anglaise, le vocabulaire et la préparation TOEIC
  • Tuteur intelligent : Obtenez des explications claires et des exemples pratiques
  • Aide aux exercices : Assistance pour vos exercices de préparation TOEIC

Données collectées

1. Données stockées localement

L'Extension stocke les données suivantes uniquement sur votre appareil (via Chrome Storage) :

  • Clé API Gemini : Votre clé API personnelle pour accéder aux services d'intelligence artificielle de Google
  • Préférences utilisateur : Paramètres de l'extension (intervalle, type d'exercice sélectionné)
  • Clé de licence : Pour vérifier votre autorisation d'utilisation
  • Identifiant d'appareil : Un identifiant unique généré localement pour la gestion des licences

2. Données transmises à des serveurs tiers

  • Google Gemini API : Le contenu des exercices (textes, questions) est envoyé à l'API Google Gemini pour analyse. Consultez la politique de confidentialité de Google pour plus d'informations.
  • Serveur de licences : Nous transmettons uniquement :
    • Votre clé de licence
    • L'identifiant unique de votre appareil
    • Aucune donnée personnelle identifiable n'est collectée

Données NON collectées

Nous ne collectons PAS :

  • Vos informations personnelles (nom, email, adresse)
  • Vos identifiants GlobalExam
  • Votre historique de navigation
  • Vos résultats aux exercices
  • Aucune donnée analytique ou de tracking

Utilisation des données

Les données sont utilisées exclusivement pour :

  • Fournir une assistance IA pendant vos exercices TOEIC
  • Vérifier la validité de votre licence d'utilisation
  • Sauvegarder vos préférences localement

Stockage et sécurité

  • Toutes les données sont stockées localement sur votre appareil
  • Les communications avec les serveurs sont chiffrées (HTTPS)
  • Le cache de licence est chiffré localement
  • Aucune base de données centralisée ne stocke vos informations personnelles

Partage des données

Nous ne vendons, n'échangeons ni ne transférons vos données à des tiers, sauf :

  • Google (via l'API Gemini) pour le traitement IA
  • Notre serveur de licences (données minimales de vérification uniquement)

Vos droits

Vous pouvez à tout moment :

  • Supprimer vos données : Désinstallez l'extension ou effacez les données via les paramètres Chrome
  • Révoquer l'accès API : Supprimez votre clé API depuis l'interface de l'extension
  • Désactiver la licence : Utilisez le bouton de désactivation dans l'onglet Licence

Permissions de l'extension

L'extension requiert les permissions suivantes :

  • activeTab : Interagir avec la page GlobalExam active
  • storage : Sauvegarder vos préférences localement
  • scripting : Injecter le script d'assistance sur GlobalExam
  • host_permissions : Communiquer avec l'API Gemini et le serveur de licences

Modifications de cette politique

Nous nous réservons le droit de modifier cette politique. Les changements seront reflétés par la date de "dernière mise à jour" ci-dessus.
